the body tag contains the ng-app. this means that the area inside of the body tag is controller by Angular. the ng-app can only be used once per page, so it is quite common to have the ng-app on the html element of the page, or in our case, the body element of the page. now all our angular expressions must exist withing the body tag of the page to work. if we try to use angular inside of an element that is not within the scope of the ng-app, it won't work!
